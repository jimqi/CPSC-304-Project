User Manual

Customer

Start at Login. Two options: login or register. Should register first. Will then be taken to shopping cart where you can add/delete items in cart and checkout. User can search for items based on category, name, title. After search is done if a result is returned User can input the UPC of item he wants to add and then can press add. Able to logout.

Manager

Able to add stock, generate daily sales report, and see top selling items for a certain date.
